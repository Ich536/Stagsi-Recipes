Sample Native Messaging Host Add-On For Google Chrome
=====================================================

This add-on allows to search Stagsi database from within Google Chrome via:
* the toolbar button using the current page selection as the search query
* the "stg" prefix in the omnibox (URL bar), example: "stg landscape | nature"

Used in Stagsi Cookbook: https://go.soletude.ca/stagsi/ex/OA
